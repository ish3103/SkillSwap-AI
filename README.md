# 🌌 SkillSwap AI

SkillSwap AI is a futuristic full-stack student skill exchange platform where users can connect, share skills, and interact with an AI-powered assistant.

---

## 🚀 Features

- 🔐 User Authentication
- 🤖 AI Assistant
- 🌌 Futuristic UI
- ⚡ Particle Background
- 🗄️ MongoDB Database
- 🔒 Password Hashing
- 📱 Responsive Design

---

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Other Tools
- bcryptjs
- dotenv
- particles.js

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ish3103/SkillSwap-AI
```

### Backend Setup

```bash
cd backend
npm install
node server.js
```

### Frontend

Run frontend using Live Server.
